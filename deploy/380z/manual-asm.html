ASM - 8080 Assembler

Usage

A>ASM filename (no extension - reads .ASM, writes .HEX and .PRN)

Then: A>LOAD filename to convert .HEX to .COM

Source Format

Label in column 1, instructions indented with tab/spaces.

        ORG  100H
        MVI  C,9      ;BDOS print string
        LXI  D,MSG
        CALL 5
        RET
MSG:    DB   'Hello$'
        END

Directives

ORG addrSet origin address
DB val,...Define bytes/strings
DW val,...Define words
DS countDefine storage (reserve bytes)
EQU valEquate label to value
ENDEnd of source

Common 8080 Instructions

MVI r,nLoad register with immediate
MOV r,rMove between registers
LXI rp,nnLoad register pair
LDA addrLoad A from memory
STA addrStore A to memory
ADD r / ADI nAdd
SUB r / SUI nSubtract
CMP r / CPI nCompare
ANA r / ANI nAND
ORA r / ORI nOR
JMP addrJump
JZ/JNZ/JC/JNCConditional jump
CALL addrCall subroutine
RETReturn
PUSH/POP rpStack operations
INR/DCR rIncrement/decrement
INX/DCX rpInc/dec register pair

Registers

A, B, C, D, E, H, L, M (memory via HL)
Pairs: BC, DE, HL, SP
