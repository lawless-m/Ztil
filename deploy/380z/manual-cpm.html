CP/M 2.2 Quick Reference

Commands

DIRList files on current drive
DIR B:List files on drive B
TYPE fileDisplay text file contents
ERA fileDelete a file
REN new=oldRename a file
B:Switch to drive B
STATShow disk status
PIP d=sCopy file (PIP B:COPY.TXT=A:ORIG.TXT)

Running Programs

Type the program name without .COM extension:
A>MBASIC runs MBASIC.COM
A>SARGON runs SARGON.COM

Page Zero

0000hWarm boot (JP to BIOS)
0005hBDOS entry point
005ChDefault FCB 1
006ChDefault FCB 2
0080hCommand tail / DMA buffer
0100hTPA start (.COM files load here)

BDOS Functions (CALL 5)

C=0System reset
C=1Console input
C=2Console output (E=char)
C=9Print string (DE=addr, $-terminated)
C=12Return version (HL=0022h)
C=15Open file (DE=FCB)
C=16Close file
C=20Read sequential
C=21Write sequential
C=22Make file
