ED - CP/M Line Editor

Starting

A>ED filename.ext

If the file exists, ED loads it. Otherwise: NEW FILE.

Commands (at * prompt)

I - Insert mode. Type lines. End with Ctrl-Z.
nT - Type n lines (T alone = current line)
nL - List n lines
B - Move to beginning
# - Move to end
nU - Move up n lines
nD - Delete n lines
nK - Kill (delete) n characters
Stext^Z - Search for text
nFtext^Zreplace^Z - Find and replace
E - Save and exit
H - Save and re-edit (keep editing)
Q - Quit without saving

Example

A>ED TEST.ASM
NEW FILE
*I
  ORG 100H
  MVI C,9
  LXI D,MSG
  CALL 5
  RET
MSG: DB 'HI$'
  END
^Z
*E
