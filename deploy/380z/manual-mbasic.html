MBASIC Quick Reference

Starting

A>MBASIC — start BASIC interpreter
A>MBASIC PROG — load and run PROG.BAS

Editing

Type line numbers to enter program lines:
10 PRINT "HELLO"
20 FOR I=1 TO 10
30 PRINT I*I
40 NEXT I

LIST — show program
RUN — execute program
NEW — clear program

File Operations

SAVE "name" — save to disk
LOAD "name" — load from disk
FILES — list .BAS files
KILL "name.bas" — delete file
SYSTEM — exit to CP/M

Common Statements

PRINTOutput text/values
INPUTRead user input
IF...THENConditional
FOR...NEXTLoop
GOTO / GOSUBJump / subroutine
DIMDeclare array
DEF FNDefine function
REMComment
DATA / READInline data
PEEK / POKEMemory access
INP / OUTPort I/O
