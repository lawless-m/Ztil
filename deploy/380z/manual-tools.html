CP/M Utilities

STAT - Disk Status

A>STAT - show drive status and free space
A>STAT *.* - show all file sizes
A>STAT DSK: - drive characteristics

PIP - File Copy

A>PIP dest=source
A>PIP B:FILE.TXT=A:FILE.TXT - copy between drives
A>PIP COPY.COM=ORIG.COM - copy on same drive
A>PIP CON:=FILE.TXT - type file to console

DUMP - Hex Dump

A>DUMP filename - show file as hex bytes

LOAD - HEX to COM

A>LOAD filename - convert .HEX to .COM
(Used after ASM to make executable)

DDT - Debugger

A>DDT filename.COM - load and debug
-D100 - dump memory at 100h
-L100 - disassemble at 100h
-G100 - execute from 100h
-S100 - set memory at 100h
-X - examine registers
-G0 - exit to CP/M

CLAUDE - Ask Claude AI

A>CLAUDE
Type your question, get Claude's response.
Uses the WebSocket connection to the host server.

Development Workflow

1. ED PROG.ASM - write source
2. ASM PROG - assemble to .HEX
3. LOAD PROG - convert to .COM
4. PROG - run it
5. DDT PROG.COM - debug if needed
